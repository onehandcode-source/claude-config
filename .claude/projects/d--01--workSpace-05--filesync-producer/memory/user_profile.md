한국 공공기관(safepeople) 소속 백엔드 개발자. Java/Spring Boot 중심의 서버사이드 개발이 주 업무이며, React(MUI) 프론트엔드도 다룬다. 네트워크 간 파일 동기화 시스템(인터넷망 ↔ 업무망 ↔ 백업망)을 개발하고 있다.

**기술 스택**: Java 8, Spring Boot 2.7, JPA/Hibernate, RabbitMQ, Tibero 7(Oracle 호환), Spring WebFlux/WebClient, Logback

**커뮤니케이션**: 한국어로 소통. 간결하고 직접적인 답변 선호. 설명보다 실행 결과 확인을 중시함("직접 run 해서 로그 확인해"처럼 결과 기반으로 진행).
