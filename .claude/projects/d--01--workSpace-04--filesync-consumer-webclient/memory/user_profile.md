## 역할

Java Spring Boot / RabbitMQ 기반 파일 동기화 시스템의 개발 및 운영 담당자.
로그 분석, 장애 분석, 서킷브레이커/메시지큐 설계 등 운영 인프라에 깊이 관여.

## 업무 스타일

- 장애 발생 시 로그 파일을 직접 제공하거나 로그 경로를 안내하며 원인 분석 요청
- 분석 결과를 타임라인/표 형식으로 정리해주면 선호
- 기술 용어(서킷브레이커, DLQ, NACK, HANG 등)를 자연스럽게 사용 — 별도 설명 불필요
- 이슈 보고서, 설계 문서, 로그 분석 가이드 등 docs/ 하위에 마크다운으로 관리
