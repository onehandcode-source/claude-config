`spring.codec.max-in-memory-size` 설정이 GET 전용 단순 프록시에서 불필요하다는 설명을 사용자가 수용함(2026-03-10).

**Why:** 이 프로젝트는 실용적 판단을 선호. 불필요한 설정은 명확히 지적해도 됨.

**How to apply:** 설정 추가 제안 시 "정말 필요한가?" 먼저 검토하고 불필요하면 직접 말할 것. 과잉 설정 방지.
